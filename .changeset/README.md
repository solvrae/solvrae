# Changesets

This folder is managed by [changesets](https://github.com/changesets/changesets).
Add a changeset for every user-facing change:

```bash
pnpm changeset
```

Pick the affected packages and a bump type (patch / minor / major), then write a
short summary. Changesets aggregates these into release PRs and changelogs.

See the [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md).
